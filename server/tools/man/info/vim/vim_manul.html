3 查与替换

3.1 简单查找

/       向下查找
？       向上查找
n       重复

.*[]^%/\?~$ 这些字符有特殊的意思，会用来模式匹配的，所以你如果要用的话，需要转义，在字符前加\

3.2 复杂查找

:set ignorecase                 忽略大小写
:set noignorecase               与上面相反
\c                              临时忽略大小写
\C                              与上面相反
*                               前面的单位重复0或多次 
\+                              1次或多次
\=                              0次或1次
\ {n,m}                         n到m次
\|                              或
[0−9a−f]                        数字0-9，字母a-f

\d      数字              [0−9]
\D      非数字             [^0−9]
\s      空白字符            [ ] ( <Tab> and  <Space> )
\S      非空白字符           [^ ] (not  <Tab> and  <Space> )

3.3 替换

:[range]substitute/from/to/[flags]
[flags]     p(print)打印替换过的行
            c(confirm)替换前确认
            g一行内所有的匹配，默认是一行的第一次匹配
[range]     1,$         表示从第一行到最后一行，或用%

6 折叠

所有折叠的命令以z开头 zf (fold) 折叠的操作符 zfap 手动创建一个折叠，折叠一个段落 zo (open) 打开一个折叠 zc (close) 关闭一个折叠 zr 减少折叠 zm 增加折叠

zM              关闭所有的折叠（适用于层叠关系）

7 撤销和恢复

u           (undo)撤销
CTRL−R      (redo)恢复
g−          以下图的这棵树的顺序来撤销
g+          以下图的这棵树的顺序来恢复

vim 会记录一个操作的历史记录，或形成一棵树，因此undo和redo只会在一条分支上，要想跨分支得用[g-]

14 关于语言、编码等问题

我们需要注意的几个选项

    :language
    set encoding
    set fileencoding
    set fileencodings

14.1 语言

当启动vim时，它会根据操作系统环境来设置语言，我们可以通过命令

    :language

来查询当前的语言，这个选项的主要作用是：设置vim的各种显示，在unix 上，我们可以这样来设置

    :language en_US.UTF-8
    :language zh_CN.UTF-8

14.2 编码

encoding 指定的是vim自身内部编码，比如菜单、各种提示、缓冲区。 fileencoding 指的是默认的文件的编码 fileencodings 文件编码候选，如果你打开的文件不是fileencoding，那么就会从这里去找相应的编码，分先后次序。

    ga      查看当前光标所在的字符的编码数值

如果vim不能自动识别编码，你需要手动的来指定，比如

    :edit ++enc=gbk −r gbk.txt
